# Evaluacion de Impacto Comunicacional

Usar esta referencia para mejorar la probabilidad de que un carrusel conecte con su audiencia desde el diseño. No garantiza resultados, pero ordena decisiones visuales, cromáticas y estructurales.

Aplicar en cada paso donde hay una decisión visual: estructura de láminas (paso 4), asignación de imágenes (paso 5), paleta (paso 6), producción (paso 9). Nunca reportar al usuario.

## Principio

Todo carrusel debe responder — desde el diseño — cuatro preguntas:

- ¿La jerarquía visual habla a la persona correcta?
- ¿La portada crea tensión visual (imagen, tamaño, contraste)?
- ¿Cada lámina tiene un foco visual único?
- ¿El cierre tiene peso visual suficiente para generar acción?

## Checklist

- Audiencia: ¿la tipografía, densidad y tono cromático se corresponden con la persona a quien va dirigido?
- Tension: ¿la portada abre una brecha visual — imagen, contraste, tamaño de titular — que invita a seguir?
- Claridad: ¿cada slide tiene un único elemento dominante?
- Carga cognitiva: ¿la densidad visual permite escanear la lámina en 3 segundos?
- Escaneo: ¿la jerarquía tipográfica y el contraste guían el ojo sin esfuerzo?
- Cercania: ¿la imagen asignada humaniza el mensaje sin dramatizarlo?
- Credibilidad: ¿los datos o citas tienen tratamiento visual diferenciado (tamaño, posición, estilo)?
- Accion: ¿el cierre tiene diferenciación visual suficiente — color, peso, espacio — para indicar que hay un paso siguiente?
- Marca: ¿colores, tipografías y composición respetan el sistema de diseño del cliente?

## Scoring rapido

Puntuar de 1 a 5 si el proyecto lo amerita:

- Claridad.
- Relevancia para audiencia.
- Fuerza del hook visual.
- Carga cognitiva.
- Cercania emocional de la imagen.
- Peso visual del cierre.
- Alineacion con marca.

Si algun criterio queda bajo 3, ajustar antes de avanzar al siguiente paso.
